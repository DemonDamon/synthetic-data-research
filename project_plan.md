# Synthetic Data Research Repository - 项目规划

## 仓库名称
**synthetic-data-research** (合成数据研究)

## 项目目标
构建一个系统化的合成数据知识体系仓库,从理论基础到实践应用,为初学者和研究者提供全方位的学习路径。

## 目录结构规划

### 第一层级:主题分类
基于已有文档,规划以下主要研究方向:

1. **01-理论基础** (Theoretical Foundations)
   - 生成对抗网络 (GANs)
   - 变分自编码器 (VAEs)
   - 扩散模型 (Diffusion Models)
   - 其他生成模型 (SMOTE, 物理仿真等)

2. **02-核心方法** (Core Methods)
   - 深度生成模型
   - 统计方法
   - 基于规则的方法
   - 混合方法

3. **03-应用领域** (Application Domains)
   - 计算机视觉
   - 自然语言处理
   - 时间序列
   - 表格数据
   - 多模态数据

4. **04-关键技术** (Key Technologies)
   - 隐私保护 (差分隐私)
   - 小样本学习
   - 域随机化
   - Sim2Real技术
   - 可控生成

5. **05-质量评估** (Quality Evaluation)
   - 保真度评估
   - 多样性评估
   - 实用性评估
   - 隐私性评估

6. **06-前沿趋势** (Emerging Trends)
   - LLMs与合成数据
   - 世界模型
   - 跨模态生成
   - 物理AI

7. **07-伦理与风险** (Ethics and Risks)
   - 偏见问题
   - 数据完整性
   - 隐私泄露
   - 缓解策略

8. **08-工业实践** (Industry Practices)
   - NVIDIA案例
   - 其他企业实践
   - 开源工具
   - 最佳实践

9. **09-学术资源** (Academic Resources)
   - 论文列表
   - 数据集
   - 基准测试
   - 会议与期刊

## 文件组织规范

### 每个子目录包含:
- `README.md` - 该主题的概览和导航
- `overview.md` - 详细的综述报告
- `papers.md` - 相关论文列表
- `resources.md` - 工具、代码、数据集等资源
- `images/` - 图片资源文件夹

### 最内层目录包含:
- `README.md` - 该细分主题的总结
- `detailed_report.md` - 深度研究报告
- `technical_notes.md` - 技术笔记
- `case_studies.md` - 案例研究
- `images/` - 相关图片

## 根目录README结构
1. 项目简介
2. 思维导图 (Mermaid格式)
3. 快速导航
4. 学习路径建议
5. 贡献指南
6. 许可证

## 调研深度策略
- 基于已有文档进行结构化整理
- 针对每个主题进行深度文献调研
- 补充最新研究进展 (2024-2026)
- 添加实践案例和代码示例
- 构建知识图谱和关联关系
